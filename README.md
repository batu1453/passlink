## Usage

### Create a URL with password

1. Open [PassLink](https://passlink.macr.app).

2. Input URL and password.

3. Click generated URL to copy.

4. Share copied URL.


### Open a URL with password

1. Open a URL.

2. Enter password.

3. Click **GO** button.
